My Work

Below are selected projects that illustrate how I merge research, design, and code to create meaningful experiences.

Redesigning Radiology Workflow

Context: NHS × RCA collaboration

Challenge: Radiologists and technicians struggled with inefficient handoffs and poor feedback systems.

Approach: Mapped existing workflows, ran co‑design sessions, and prototyped a visual dashboard that simplified case tracking.

Outcome: Improved communication loops, reduced delays in image review, and fostered a stronger collaboration culture.

Skills: UX research, service blueprinting, prototyping

Addressing Nurse Burnout

Context: Healthcare service design initiative

Challenge: Burnout among nurses undermined patient care and morale.

Approach: Conducted ethnographic interviews, mapped stress triggers, and co‑developed a wellbeing toolkit integrated into shift planning.

Outcome: A proposed framework adopted by local clinics; satisfaction scores improved during the pilot program.

Skills: Systems thinking, qualitative research, journey mapping

Local‑to‑Local Platform (Amsterdam)

Context: Urban community service design initiative

Challenge: Strengthen the circular economy between local producers and citizens.

Approach: Designed a service blueprint and digital prototype for shared‑value interactions between stakeholders.

Outcome: Concept recognized by sustainability innovation councils and showcased at academic conferences.

Skills: Service design, co‑creation, UX wireframes

Neurodiversity Justice Platform

Context: Inclusion‑driven non‑profit project

Challenge: Create accessible tools for neurodiverse individuals interacting with justice systems.

Approach: User research with affected groups and iterative prototyping for readability and tone.

Outcome: Increased comprehension of rights content by 40% during user testing.

Skills: Accessibility design, UX writing, front‑end prototyping

Portfolio Coding System

Context: Personal development

Challenge: Merge service design storytelling with real code implementation.

Approach: Built modular components in Framer and React; explored visual design systems for responsiveness.

Outcome: Unified professional portfolio site demonstrating both design sensibility and technical fluency.

Skills: UI development, design systems, React
